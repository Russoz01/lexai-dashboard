### B) Anti-alucinação HARD (vinculante)

`buildAntiHallucinationFooter()` apenda em todos os agentes:
- Acórdão: nunca inventa → escreve `[acórdão a verificar]`
- Artigo de lei: cita só o que tem certeza absoluta (vade mecum) → senão `[verificar redação atual]`
- Súmula: `[súmula correlata a confirmar]` quando incerto
- Doutrina: `doutrina majoritária entende` quando não souber autor específico
- Fato concreto: `[INFORMAÇÃO A COMPLETAR PELO ADVOGADO]`
- LGPD: zero dados pessoais identificáveis no output
- OAB: zero linguagem de captação ilícita (Provimento 205/2021)

### C) User preferences injection

`buildPreferencesContext(prefs)` lê `user_preferences` e inject como bloco system **sem cache** (varia por user):
- `prefs.tom` (parceiro/profissional/casual) calibra estilo
- `prefs.area_juridica_padrao` orienta exemplos/doutrina/jurisprudência
- `prefs.idioma` (pt-BR padrão)

Cache_control ephemeral do SYSTEM_PROMPT preservado intacto — cache hits da Anthropic seguem economizando 80% input tokens.

### D) Memória cross-agent

`recordAgentMemory()` fire-forget após cada resposta:
- `resumo`: `[agente] entrada → saída` (cap 500 chars)
- `fatos`: array de pares chave/valor (área, tipo, score, etc.)
- `tags`: heurística com `extractMemoryTags()` (slug + área + 13 keywords jurídicas)

`memory_enabled = false` ⇒ no-op silencioso.
TTL 90 dias automático na tabela.

### E) Chat orquestrador com smart suggestions

`/api/chat` agora:
1. Lê `getRecentMemory(usuario, limit:5)` quando `prefs.smart_suggestions && prefs.memory_enabled`
2. Injeta `formatMemoryForPrompt(memory)` no system prompt

4. Persiste memória do próprio chat (mensagem + agente_rotado)

## Antes vs depois (exemplo: chat orquestrador)

```ts
function buildSystemPrompt(
  fidelidade: Fidelidade,
  memoryContext: string = '',          // NOVO
  smartSuggestionsHint: string = '',   // NOVO
): string {
  return `... ${tomPorFidelidade(fidelidade)} ... HUMANIZAÇÃO ... ${memoryContext}${smartSuggestionsHint}`
}

// Pré-LLM:
let memoryContext = ''
let smartSuggestionsHint = ''
let prefsContext = ''
if (usuarioIdEarly) {
  const prefs = await getUserPreferences(supabase, usuarioIdEarly)
  prefsContext = buildPreferencesContext(prefs)
  if (prefs.smart_suggestions && prefs.memory_enabled) {
    const memory = await getRecentMemory(supabase, usuarioIdEarly, { limit: 5 })
    memoryContext = formatMemoryForPrompt(memory)
    if (memory.length > 0) {
      smartSuggestionsHint = `\n\nUSE A MEMÓRIA RECENTE PRA CONTEXTUALIZAR. Se o usuário continuar trabalho de uma interação anterior...`
    }
  }
}

system: [
  { type: 'text', text: buildSystemPrompt(fidelidade, memoryContext, smartSuggestionsHint), cache_control: ... },
  ...(prefsContext ? [{ type: 'text', text: prefsContext }] : []),
]
```
